# 끝말잇기 게임 🎮

React로 만든 끝말잇기 단어 게임입니다.

## 📌 프로젝트 소개
시작 단어가 주어지면 마지막 글자로 시작하는 단어를 이어가는 게임입니다.
엔터키 또는 확인 버튼으로 단어를 입력할 수 있습니다.

## ⚙️ 주요 기능
- 시작 단어 props 전달
- 첫 글자 / 마지막 글자 일치 여부 검증
- 정답 / 오답 안내 메시지 표시
- 엔터키 입력 지원
- 자동 포커스

## 🛠 사용 기술
- React (useState, useEffect, useRef)
- Vite
- SCSS

## 🚀 실행 방법
```bash
npm install
npm run dev
```
